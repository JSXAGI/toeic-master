# TOEIC® Master 300+

TOEICスコアアップを目指すための、超実践的クイズアプリケーションです。
初級から最上級（Super Advanced）まで、厳選された350問を収録しています。

## 🚀 主な機能
- **レベル別クイズ**: Beginner / Intermediate / Advanced / Super Advanced
- **ランダム出題**: 問題順および選択肢(A-D)の配置を毎回シャッフル
- **可変タイマー**: レベルに応じた制限時間設定（30秒〜60秒）
- **苦手克服モード**: 間違えた問題をLocalStorageに保存し、集中的に復習可能
- **レスポンシブデザイン**: PC・スマートフォンの両方に対応

## 🛠 使用技術
- HTML5 / CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## 📖 使い方
1. `index.html` をブラウザで開きます。
2. 挑戦したいレベルを選択します。
3. 制限時間内に回答を選択してください。
